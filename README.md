# 3ds Max V-Ray 材质工具箱（Material Toolkit）

一个**超轻量**的 3ds Max 脚本（MaxScript），把场景里的 V-Ray 材质批量替换成普通材质（Standard / Physical），并可以把所有材质的「构成」导出成一份对 AI 友好的 Markdown 文档，方便你丢给 ChatGPT / Claude 直接生成可视化 HTML 报告。

没有编译、没有外部依赖、不用装插件管理器——复制粘贴就能跑，稳定性优先。

---

## ✨ 功能特性

- **一键替换 V-Ray 材质**：支持 `VRayMtl`、`VRayLightMtl`、`VRayMtlWrapper`、`VRayBlendMtl`、`VRay2SidedMtl`、`VRayOverrideMtl`、`VRayFastSSS2` 以及其它所有以 `VRay` 开头的材质，统一转成 Standard（或 Physical）材质。
- **尽量保留外观**：可勾选「保留漫反射 / 凹凸 / 透明贴图」，并顺手把 `VRayBitmap` / `VRayHDRI` 转成普通 `BitmapTexture`。
- **彻底去 V-Ray 依赖**：多维子材质、嵌套材质会递归处理，场景里所有引用（含材质编辑器 24 槽位、环境）一次性替换，不漏。
- **可撤销**：替换操作包了 `undo`，出错随时 `Ctrl+Z`。
- **导出材质构成 MD**：把场景中每个材质的类型、使用对象、颜色、数值参数、贴图、子材质关系，导出成结构化 Markdown。
- **自带 AI 提示词**：导出的 MD 开头内嵌一份「MD → HTML 报告」的转换提示词（HTML 注释），AI 读到就能渲染；面板里也有「复制 AI 提示词」按钮。

---

## 📋 环境要求

- **3ds Max**（任意较新版本均可；Physical 材质目标需要 3ds Max 2017+）
- 场景里**原本有 V-Ray 材质**即可（本工具不要求本机安装 V-Ray 也能跑替换，但导出的材质属性来自场景已有数据）

## 🚀 安装与使用

### 1. 安装（两种方式任选）

- **方式 A**：把 `vray_material_replacer.ms` 直接**拖进 3ds Max 视口**，自动运行并弹出工具箱窗口。
- **方式 B**：菜单 `MAXScript → 新建脚本`，打开该文件，按 `Ctrl+E` 运行。

### 2. 面板一：V-Ray 材质替换

工具箱弹窗中的第一个面板：

1. **目标材质类型**：选 `Standard (标准材质)` 或 `Physical (物理材质)`。
2. 勾选 **保留漫反射/凹凸/透明贴图**（默认勾选）。
3. 如需后续用扫描线 / 物理材质渲染，勾选 **把 VRayBitmap / VRayHDRI 转为 Bitmap**。
4. 点 **「开始替换 V-Ray 材质」**，底部日志会显示替换数量。

> 💡 替换前先 **保存场景**（脚本只改内存中的材质，但养成习惯更稳妥）。

### 3. 面板二：导出材质构成 MD

工具箱弹窗中的第二个面板：

1. 设置**输出路径**（默认会放在当前场景同目录，名为 `材质构成报告.md`）。
2. 三个勾选项按需选择：**使用对象** / **贴图** / **数值·颜色参数**（默认全开）。
3. 点 **「导出材质构成 MD」** → 自动生成文件并打开所在文件夹。
4. 需要把 MD 转 HTML 时，点 **「复制 AI 提示词」** 把指令复制到剪贴板，连同导出的 MD 一起发给 ChatGPT / Claude 即可。

> 想先看看最终结构？仓库里 `examples/材质构成报告_示例.md` 是一份假数据示例，不用开 Max 就能拿去试 AI → HTML。

## ⚠️ 已知限制

- **V-Ray 专属参数会被丢弃**：反射 / 折射 / 光泽度 / Fresnel / SSS 这些在标准材质里没有对应项，按「越简单越稳定」原则直接丢弃。结果是「能正常打开、无 V-Ray 依赖」，而不是「渲染效果完全一致」。
- **`VRayLightMtl` 的强度**不精确映射到标准自发光强度（标准自发光没有 HDR 级别亮度）；颜色对、亮度大致对，极亮的灯可能不够亮。
- **V-Ray 贴图（如 `VRayDirt`、`VRayColor`）原样保留**：在标准 / 物理材质下可能渲染成黑，但不影响稳定性。
- **替换是单向的**：没有「撤销替换还原 V-Ray」的反向功能（可用 `Ctrl+Z` 回退本次操作）。
